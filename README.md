# WhatNext Vision Motors
Shaping the Future of Mobility with Innovation and Excellence

WhatNext Vision Motors is implementing a Salesforce-based solution to improve customer experience, streamline the ordering process, and enhance operational efficiency. This project focuses on automating dealer assignment, validating vehicle stock, managing orders accurately, and improving customer communication through Salesforce automation tools.

## Project Overview
This Salesforce project enhances the companyâ€™s vehicle ordering and customer management workflows. It introduces automated dealer assignment based on customer location, prevents customers from ordering out-of-stock vehicles, and updates order statuses automatically. Scheduled processes and batch jobs ensure that stock levels and order records remain accurate and current.

## Features and Requirements

### Salesforce CRM Implementation
Store and manage vehicle details, dealer information, and stock availability within Salesforce.
Track customer orders, test drives, and service requests.

### Process Automation
Prevent order creation for out-of-stock vehicles.
Automatically assign the nearest dealer based on customer location.
Send automated email reminders for upcoming test drives.

### Apex and Triggers
Use Apex triggers to validate stock levels and assign dealers.
Follow best practices using trigger handlers for modular and maintainable code.
Automatic updates to order status depending on stock availability.

### Batch Apex
Create batch Apex jobs to update stock levels on a scheduled basis.
Process pending orders based on updated inventory.
Send automated notifications related to order and stock changes.

### Scheduled Apex
Run scheduled processes that handle stock checking, order status updates, and routine automation.
